## LinkedList
A linked list is a linear data structure where elements are stored in nodes. Each node contains:

- A value (the data it holds).
- A reference (or pointer) to the next node in the sequence.

Linked lists are dynamic, meaning they can grow or shrink as needed, unlike arrays with fixed sizes.

### Types of LinkedList:

There are 3 core types:

1. Singly Linked List — each node points only forward
2. Doubly Linked List — each node points forward and backward
3. Circular Linked List — last node points back to the head (can be singly or doubly)
